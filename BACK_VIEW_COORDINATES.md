## HOTSPOT NAME MAPPINGS

### Front View
- LD1 → Left Stick
- LB → Left Bumper
- LD2 → D Pad Up
- LD3 → D Pad Left
- LD4 → D Pad Down
- LD5 → D Pad Right
- RB → Right Bumper
- RD1 → Button Y
- RD2 → Button B
- RD3 → Button X
- RD4 → Button A
- RD5 → Right Stick

### Back View (New Names)
- LD1 → **RP** (Right Paddle 2)
- LD2 → **RT** (Right Trigger)
- LD4 → **SR** (Shoulder Right)
- RD1 → **LT** (Left Trigger)
- RD4 → **SL** (Shoulder Left)
- RD5 → **LP** (Left Paddle 2)

## TOOLTIP LABEL MAPPINGS

### Back View Tooltip Labels
- leftStick (LD1/RP): "Paddle" / "Right Paddle 2"
- dPadUp (LD2/RT): "Trigger" / "Right Trigger"
- dPadDown (LD4/SR): "Button" / "Shoulder Right"
- buttonY (RD1/LT): "Trigger" / "Left Trigger"
- buttonA (RD4/SL): "Button" / "Shoulder Left"
- rightStick (RD5/LP): "Paddle" / "Left Paddle 2"
